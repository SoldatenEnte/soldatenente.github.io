Neo Sliders

A Minimalist Gravity-Flipping Speedrunner

← Back to Portfolio

Project Overview

The vision for Neo Sliders is to create a skill-based arcade game that is easy to learn but difficult to master. By reducing player control to a single mechanic—flipping gravity—the gameplay focuses entirely on mastering momentum to speedrun through challenging, curved levels.

Gallery

Tech Stack

  • Unity
  • C#

Key Features

  • Core Gravity Mechanic: A one-touch control system allows the player to instantly change the direction of gravity.
  • Physics-Based Movement: The ball's movement is physics-driven, requiring the player to build and maintain momentum to navigate tracks efficiently.
  • Level Pack: The game includes 8 unique, handcrafted levels designed to test the player's control and speed.
  • Speedrunning System: A timer tracks the current run, and a "Best Time" is saved for each level to encourage replayability.
  • Polished Visuals: A cohesive neon art style gives the game a distinct, minimalist look.
