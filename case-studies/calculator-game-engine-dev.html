Project Overview

This entry chronicles a multi-year personal challenge, born within the locked-down environment of a school-issued iPad: to push an enhanced, emulated TI-83 graphing calculator far beyond its intended purpose. Driven by pure curiosity, I set out to transform this unique tool into a capable game development environment by engineering solutions to problems that most developers take for granted, sparking a deep dive into the emulator's architecture.

Tech Stack

  • TI-BASIC

Key Features & Discoveries

  • Custom Game Engine: Designed and built a rudimentary engine in TI-BASIC capable of running both turn-based and real-time games, achieving real-time action with a processor-bound loop to approximate frame rates.
  • A Library of Games: The engine was used to create a portfolio of fully playable games that became popular among fellow students, including _Chess_, a complete version of _2048_ with a self-playing AI, and a functional clone of _Flappy Bird_.
  • System Vulnerability Discovery: Through deep analysis, I discovered a critical flaw in the emulator's save-state system, leading to an exploit where a controlled crash could bypass all exam security measures undetected.
