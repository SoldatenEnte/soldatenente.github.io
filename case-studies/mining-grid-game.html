Mining Grid Game

A Procedural 2D Platformer

Project Overview

The vision for this project is to create a 2D sandbox mining game inspired by classics like _Terraria_, where players can explore, mine, and shape a procedurally generated world. The goal is to provide endless replayability by generating unique and unpredictable cave systems and landscapes for the player to discover with each new game.

Tech Stack

  • Godot Engine
  • GDScript

Key Features & Explored Concepts

  • Procedural World Generation: Explored algorithms to generate a tile-based, destructible world, creating unique landscapes and cave systems on the fly.
  • Dynamic Tile System: Focused on techniques for advanced sprite manipulation, allowing for tiles to dynamically update their appearance based on adjacent blocks (autotiling) as the player mines through the terrain.
  • 2D Platformer Physics: Implemented core character controls for movement and interaction within the grid-based environment.
