Project Overview

This project is a real-time 3D web simulation of a motorized 3-axis Stewart platform, developed to visualize and test inverse kinematics algorithms before deployment on physical hardware. The simulator, built with React and Three.js (React Three Fiber), allows users to control the platform's roll and pitch angles or manipulate individual actuator heights. It accurately calculates the required actuator positions to achieve the desired orientation, providing a crucial tool for debugging and demonstrating the complex mathematics involved. An interactive control panel powered by Tweakpane offers real-time manipulation of all system parameters.

Key Features

  • Real-time 3D visualization of a 3-axis Stewart platform and its components.
  • Accurate inverse kinematics simulation converting target roll/pitch angles into individual actuator heights.
  • Interactive control panel (Tweakpane) for manipulating platform radius, actuator distance, target angles, and manual actuator heights.
  • Dual control modes: "Target Angle" for kinematic simulation and "Manual" for direct actuator control.
  • A physics-based ball simulation on the platform's surface to visually demonstrate stability and tilt.
  • Includes a separate single actuator simulation to demonstrate its individual motion and homing sequence.

Challenges & Solutions

The primary technical challenge was implementing the inverse kinematics algorithm to accurately translate desired roll and pitch angles into the precise vertical heights for each of the three actuators in real-time. This was solved by deriving the geometric equations within a React Three Fiber useFrame hook, which runs on every rendered frame. The algorithm calculates a target platform plane based on the input roll/pitch, then solves for the intersection points of the vertical actuator axes with this plane to determine the required heights. This ensures the 3D model's movement is a faithful and performant representation of the mathematical model.
